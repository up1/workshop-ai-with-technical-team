# AI Agent for Data Analysis
* langchain
* csv
* faiss-cpu
* pandas

## 0. Install libraries
```
$pip install -r requirements.txt
```

## 1. Run agent

Setup with [OpenAI API Key](https://platform.openai.com/api-keys)
```
$export OPENAI_API_KEY=<your api-key>
```

Run
```
$python demo-agent.py
```
