# Demo :: Simple chatbot
* [Streamlit](https://docs.streamlit.io/)
* OpenAI
* Anthropic

## Install
```
$pip install -r requirements.txt
```

## Run with streamlit :: ChatBot
* OpenAI
```
$streamlit run --server.maxUploadSize=10 app-qa.py
```
